## 4. 方案设计

### 4.1 桌宠窗口

使用 `PySide6` 创建无边框透明窗口：

```python
class PetWidget(QWidget):
    def __init__(self):
        super().__init__()
        self.setWindowFlags(
            Qt.FramelessWindowHint |
            Qt.WindowStaysOnTopHint |
            Qt.Tool
        )
        self.setAttribute(Qt.WA_TranslucentBackground)
        self.setFixedSize(128, 128)
```

**窗口特性：**
- 透明背景
- 始终置顶
- 可拖拽移动
- 吸附屏幕边缘

### 4.2 动画引擎

**静态图 + 代码动画：**

```python
animations = {
    'idle': {
        'y_offset': 'sin(t*2)*5',
        'blink': 'every 3s for 0.1s'
    },
    'working': {
        'x_offset': 'sin(t*3)*3'
    },
    'focus': {
        'scale': 0.8,
        'opacity': 0.5
    },
    'stretch': {
        'scale_y': '1 + sin(t*4)*0.2'
    },
    'exhausted': {
        'rotation': -15,
        'x_offset': 'random(-2,2)'
    },
    'break': {
        'rotation': 'sin(t)*10',
        'scale': 1.1
    },
    'happy': {
        'y_offset': 'abs(sin(t*5))*20',
        'scale': 1.2
    }
}
```
