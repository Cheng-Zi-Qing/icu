### 4.4 Ollama 集成

**Prompt 模板：**

```python
system_prompt = f"""
你是 {pet_name}，性格特点：{traits}。
语气风格：{tone}。
用户已经 {action_context}，请用 1 句话提醒 TA {reminder_type}。
要求：不超过 20 字，符合角色性格。
"""

# 示例
user_prompt = "用户已经盯着屏幕 20 分钟了，提醒 TA 远眺休息"
```

**调用流程：**

```python
def generate_reminder(reminder_type, context):
    if not ai_enabled:
        return random.choice(default_messages)

    try:
        response = ollama.generate(
            model=ollama_model,
            prompt=build_prompt(reminder_type, context),
            timeout=3
        )
        return response['text']
    except Exception as e:
        logger.warning(f"AI generation failed: {e}")
        return random.choice(default_messages)
```

**降级策略：**
- API 调用失败 → 使用默认文案库
- 响应超时（> 3s）→ 使用默认文案库
- 生成内容过长（> 30 字）→ 截断或重新生成
