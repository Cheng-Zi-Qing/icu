# 配置界面设计方案

## 设计原则

**独立窗口 vs 桌宠弹窗**：

✅ **采用独立窗口**（Menu Bar 触发）
- 配置项较多，需要足够空间
- 避免遮挡桌宠本体
- 符合 macOS 应用习惯

❌ **不采用桌宠弹窗**
- 空间受限
- 影响桌宠动画展示
- 交互不便

## 技术实现

### PySide6 实现

```python
from PySide6.QtWidgets import QDialog, QTabWidget

class SettingsDialog(QDialog):
    def __init__(self, parent=None):
        super().__init__(parent)
        self.setWindowTitle("I.C.U. 设置")
        self.setFixedSize(600, 500)
        self.setModal(True)  # 模态窗口

        # Tab Widget
        tabs = QTabWidget()
        tabs.addTab(BasicSettingsTab(), "基础设置")
        tabs.addTab(AIPersonaTab(), "AI 人设")
        tabs.addTab(AboutTab(), "关于")
```

### 调用方式

```python
# Menu Bar 点击"设置"时
def on_settings_clicked():
    dialog = SettingsDialog()
    if dialog.exec() == QDialog.Accepted:
        # 保存配置
        save_config(dialog.get_config())
```

## 设计优势

1. **清晰的信息架构** - Tab 分类明确
2. **符合平台习惯** - macOS 标准窗口
3. **不干扰桌宠** - 独立窗口，不遮挡动画
4. **易于扩展** - 新增 Tab 即可添加功能
